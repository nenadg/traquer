# traquer
Records and reproduces user's in-page behavior

Traquer is a tool that enables you to record a testing session on you web/page/app, than reproduce it, monitor how it performed, create a na√Øve heatmap, and even automate your testing sessions using Selenium Webdriver and Jasmine...

Original blog post about this is already here - http://lessgeneric.com/testing/traquer-testing-made-easy/ so go check it out.
